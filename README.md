# Metro-Bus ML: прогнозирование пассажиропотока автобусов при сбоях в метро

Магистерская диссертация. Разработка ML-сервиса для прогнозирования прироста пассажиропотока на автобусных маршрутах при чрезвычайных событиях на линиях метрополитена.

## О проекте

Модель по информации о сбое в метрополитене (линия, время, предполагаемая длительность) прогнозирует прирост пассажиропотока на ближайших автобусных маршрутах в ближайший час.

**Источник данных:** открытый датасет Metropolitan Transportation Authority (MTA, Нью-Йорк) с портала data.ny.gov за 2020–2025 гг.

## Стек технологий

- **Обработка данных:** pandas, numpy
- **Модели:** scikit-learn, XGBoost
- **Подбор гиперпараметров:** Optuna
- **Трекинг экспериментов:** MLflow
- **Версионирование данных:** DVC
- **Сервис:** FastAPI
- **Контейнеризация:** Docker
- **Мониторинг:** Streamlit, SQLite
